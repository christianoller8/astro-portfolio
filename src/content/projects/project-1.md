## El Desafío

El cliente necesitaba migrar de una tienda física a una plataforma digital robusta que pudiera gestionar inventario en tiempo real, procesar pagos seguros y ofrecer una experiencia de usuario fluida en dispositivos móviles. El principal reto fue integrar un sistema de geolocalización para calcular envíos de kilómetro cero.

## Mi Rol

Como Lead Developer, fui responsable de:

* Diseñar la arquitectura de la base de datos en **PostgreSQL**.
* Implementar el flujo de pagos con la API de **Stripe**.
* Desarrollar componentes reutilizables con **React** y **Tailwind**.

## Resultado

El lanzamiento de la plataforma resultó en un aumento del **150% en las ventas** durante el primer trimestre. La optimización del rendimiento (Lighthouse score de 98) aseguró una tasa de rebote mínima.

### Stack Tecnológico

* **Frontend:** Next.js 14, Framer Motion.
* **Backend:** Node.js, Prisma ORM.
* **Infraestructura:** Vercel, AWS S3.
